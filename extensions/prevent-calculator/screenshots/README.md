# Screenshots

The plugin README references two screenshots that must be captured from a
live Canvas instance (e.g. a staging/testing host where the plugin is
installed) and committed here before open-source publication:

| File | What to capture |
| --- | --- |
| `calculator-modal.png` | The PREVENT calculator modal open in the right chart pane, pre-filled from a test patient's chart (showing the per-field provenance tags). |
| `calculator-results.png` | The same modal after clicking **Calculate**, showing the six 10-year / 30-year risk scores and the "saved to chart" confirmation. |

Use a synthetic test patient only — never capture real patient data (PHI).
PNG format; crop to the modal. Once both files are added, the README image
links resolve and the open-source screenshot requirement is satisfied.
